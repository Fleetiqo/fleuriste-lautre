# L'Autre Fleur - Fleuriste

Modern website for L'Autre Fleur florist in Saint-Hippolyte, Quebec.

## Features
- Elegant floral design
- Photo gallery
- Services showcase
- Contact information
- Fully responsive

## Tech Stack
- Vite.js
- Vanilla JavaScript
- CSS3

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy
Configured for GitHub Pages with automatic deployment on push to main branch.

Site will be live at: https://fleetiqo.github.io/fleuriste-lautre/
